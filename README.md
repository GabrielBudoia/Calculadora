# Calculadora WPF

Calculadora com interface visual estilizada usando **WPF Styles e formas geométricas tracejadas**. Permite operações matemáticas básicas e algumas funções adicionais.

## 🚀 Funcionalidades
- Operações básicas: `+`, `-`, `*`, `/`
- Raiz quadrada
- Limpar: `AC` (tudo) e `C` (último valor)
- Interface estilizada com WPF (botões geométricos tracejados)

## 🛠 Tecnologias
- C#
- WPF
- Visual Studio 2022

## ▶️ Como executar
1. Abra o projeto no Visual Studio.
2. Compile o projeto (`Ctrl + Shift + B`).
3. Execute (`F5` ou "Iniciar Depuração").
4. Use a calculadora normalmente.

## 📚 Aprendizados
- Criação de interface visual customizada com **Styles e Templates**.
- Lógica de operações matemáticas implementada separadamente da interface.
- Boas práticas de UI e UX no WPF.
